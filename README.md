## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/patika-hepsiburada-react-bootcamp/odev-4-elifnurkarakoc.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run command

   ```sh
   npm start
   ```
